# Data Visualisation Analysis of Treadmill Customer Behaviour

This project analyses customer purchasing behaviour for Cardio Good Fitness, a retail business that sells treadmill products. The aim is to explore how customer characteristics differ across treadmill models and to present meaningful insights through data visualisations and dashboards.

The analysis was completed using Python in a Jupyter Notebook, with a focus on data preparation, exploratory data analysis, visualisation, dashboard creation, and business storytelling.

## Project Overview

Cardio Good Fitness collected customer data from individuals who purchased treadmills over a three-month period. The dataset includes customer details such as age, gender, education level, marital status, income, fitness level, expected treadmill usage, and expected weekly mileage.

This project explores how these factors relate to the treadmill model purchased, helping the business better understand its customer segments and product preferences.

## Dataset

The dataset used is `cardio_good_fitness.csv`.

It contains the following variables:

- Product: Treadmill model purchased
- Age: Customer age
- Gender: Customer gender
- Education: Years of education completed
- MaritalStatus: Customer marital status
- Usage: Expected treadmill usage per week
- Fitness: Self-rated fitness level
- Income: Annual household income
- Miles: Expected miles walked or run per week

## Objectives

The main objectives of this project are to:

- Prepare and clean the dataset for analysis
- Explore customer profiles across different treadmill models
- Identify patterns in customer behaviour, fitness level, income, usage, and mileage
- Create visualisations to answer business and research questions
- Build dashboards to support data storytelling and business recommendations

## Key Questions Explored

This project answers questions such as:

- Which treadmill model is purchased the most?
- What are the main age groups of treadmill customers?
- Do male and female customers show different product preferences?
- How does income affect treadmill model choice?
- Which treadmill model is linked to higher expected usage?
- How does fitness level relate to the treadmill model purchased?
- Which customers are more likely to purchase higher-end treadmill models?
